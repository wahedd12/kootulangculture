KootulangCulture

KootulangCulture is a modern, interactive Yoruba heritage quiz web application built with React and Firebase. It is designed to help users learn about Yoruba towns, culture, and their kingsâ€™ titles, as well as numbers in Yoruba, and then test their knowledge through engaging quizzes.

Mission
For all Yoruba sons and daughters around the world, KootulangCulture aims to preserve and share Yoruba heritage in a fun, educational way.

Key Features

User Authentication: Secure registration and login using Firebase Authentication.

Educational Quizzes: Multiple quizzes on Yoruba numbers and towns with kings, featuring timed questions, score tracking, and dynamic grading.

Leaderboard: Users can track their quiz history and performance over time.

Responsive Design: Fully mobile-friendly with modern UI components and gradient backgrounds.

Real-Time Data: Quiz results are stored and retrieved from Firebase Firestore in real time.

Protected Routes: Authenticated access ensures a personalized experience.
